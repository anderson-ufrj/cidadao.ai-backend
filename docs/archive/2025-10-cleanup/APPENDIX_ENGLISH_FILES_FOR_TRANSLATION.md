## Automatização Sugerida

### Script de Validação de Idioma

```python
#!/usr/bin/env python3
import os
import glob

en_keywords = ['API', 'Overview', 'Guide', 'Implementation', ...]
pt_keywords = ['Agente', 'Documentação', 'Guia', 'Implementação', ...]

for md_file in glob.glob('docs/**/*.md', recursive=True):
    with open(md_file) as f:
        content = f.read(1000)

    en_count = sum(content.count(kw) for kw in en_keywords)
    pt_count = sum(content.count(kw) for kw in pt_keywords)

    if en_count > pt_count * 1.5:
        print(f"⚠️  EN: {md_file}")
```
